Project Silent

For this project, I am developing it as the sole, independent developer, this project is still under construction, however, so far there is a physics-based rigidbody player controller, with precise control of acceleration, max speed, independent of player mass, I have also developed an abstract interaction system which allows for player grabbing of kinematic objects such as guns and physics object such as crates and bottles, it also allows for general interactions with environmental objects such as pushing a button or opening a door. This project is planned to be released in 2024.
